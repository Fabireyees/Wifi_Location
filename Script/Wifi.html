from IPython.display import HTML
import random

def hide_toggle(for_next=False):
    this_cell = """$('div.cell.code_cell.rendered.selected')"""
    next_cell = this_cell + '.next()'

    toggle_text = 'Toggle show/hide'  # text shown on toggle link
    target_cell = this_cell  # target cell to control with toggle
    js_hide_current = ''  # bit of JS to permanently hide code in current cell (only when toggling next cell)

    if for_next:
        target_cell = next_cell
        toggle_text += ' next cell'
        js_hide_current = this_cell + '.find("div.input").hide();'

    js_f_name = 'code_toggle_{}'.format(str(random.randint(1,2**64)))

    html = """
        <script>
            function {f_name}() {{
                {cell_selector}.find('div.input').toggle();
            }}

            {js_hide_current}
        </script>

        <a href="javascript:{f_name}()">{toggle_text}</a>
    """.format(
        f_name=js_f_name,
        cell_selector=target_cell,
        js_hide_current=js_hide_current, 
        toggle_text=toggle_text
    )

    return HTML(html)

Wifi Challenge - Fabi Reyes¶

Recent advances in technologys are positioning business´s effors on focusing on services where precise localization is crucial. Using machine learning as an esential tool for this, wich is why , it has been requested to analyze some data of the Jaumme´s university with 1939 observations and 520 features to predict Building ID, Floor,Latitude and Longitude.

For this task, some approaches have been considered:

  1. Cleaning the data the most without deleting important features
  2. Logaritm to lineal convertation for some features
  3. Consideration of the distance regarding "Distance and similiarity metrics"
  4. Signal strenght values-thresholding strategy regarding the near cero variation elimination
  5. PCA analysis performed for some of the predictions that perform better

The idea is try to get the best way to represent the data and based on that, predict Building ID,Floor,Latitude and Longitude. In this analysis, a previous data exploration will be show.

The metrics for evaluate the performance are: Building ID and Floor:Accuracy Latitude and Longitude: RMSE

Conclusions¶

Precision for localization is crucial, this is why been precise is fundamental for any bussiness understand it. After the cross validation analysis and choosing the best parameters for the models with tune grid for KNN, i have choose the neighbors that perform better wich are for longitude and latitude 1-N. Measures were grouped according to their similarities, euclidean distance consider for the calculation the distance similarity between 2 signals providen by the same wap wich is why we have try to dont lose information. KNN looks like performed better in almost all the predictions because of the nature of the algorithm that fits in the nature of our data.
